# SSD Hardware Research (Pre-Purchase)

Before purchasing an SSD, confirm:
- Controller model
- NAND type
- Form factor (mSATA, microSATA, M.2 SATA, 2.5\" SATA)

MP Tools are controller- and NAND-specific.

**Recommendation:** Prefer Silicon Motion (SMI) controllers.
