# High-Level Process (REDMAG)

This page describes the process as a flow, not a script.

1. Establish a baseline (known-good media)
2. Select candidate SSD hardware
3. Read device identity and capabilities
4. Apply changes (if applicable) and validate
5. Stress test and document results
