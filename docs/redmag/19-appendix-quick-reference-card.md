## Appendix: Quick Reference Card

**Print this page for your workbench!**

### Pre-Flash Checklist

- [ ] SSD connected via USB adapter
- [ ] MP Tool detects drive
- [ ] Configuration unlocked (password entered)
- [ ] Model name copied from whitelist (exact!)
- [ ] Capacity set to IDEMA mode
- [ ] Capacity matches model number
- [ ] No other drives connected
- [ ] Computer plugged into power
- [ ] Settings screenshot taken
- [ ] Ready to accept risk of bricking

### Whitelist Quick Reference

**Most Common:**
```
RED 256GB Rev T3    (256GB, one space)
RED 512GB V4        (512GB, one space)
RED 128GB Rev T3    (128GB, one space)
RED  64GB Rev T3    (64GB, TWO spaces!)
```

### SMI MP Tool Quick Start

1. Run as Administrator
2. Click "Config"
3. Password: [spacebar] [spacebar]
4. Model Name: [Copy from whitelist]
5. Capacity: Select "IDEMA"
6. Capacity: Select matching GB
7. Click "Save Config"
8. Click "START"
9. Wait for "PASS"
10. Done!
