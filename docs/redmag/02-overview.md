# Overview

This guide documents a structured workflow for preparing SSD media that is accepted by RED cameras using REDMAG.

The process involves:
- Hardware research
- MP Tool validation
- Low-level SSD configuration
- In-camera verification

This is not a supported or guaranteed process.
